## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'auth_m', :git => "https://github.com/ditec/auth_m"
```

And then execute:
```bash
$ bundle install
```

```bash
$ rails g auth_m:install
```

Or install it yourself as:
```bash
$ gem install auth_m
```
